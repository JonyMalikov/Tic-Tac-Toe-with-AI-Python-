# Крестики-нолики с ИИ (Python)

Классическая игра в крестики-нолики, реализованная на Python с искусственным интеллектом. Игра включает два уровня сложности ИИ: простой (случайные ходы) и сложный (непобедимый алгоритм Минимакс).

## Особенности

- **Классический геймплей**: Традиционная игра на поле 3x3
- **Два режима игры**:
  - Режим для двух игроков (X против O)
  - Режим игрок против ИИ
- **Уровни сложности ИИ**:
  - Простой: ИИ делает случайные ходы
  - Сложный: ИИ использует алгоритм Минимакс (непобедимый)
- **Удобный интерфейс**: Простой текстовый интерфейс с понятными инструкциями
- **Проверка ввода**: Надежная обработка ошибок при вводе пользователем
- **Отслеживание состояния игры**: Автоматическое определение побед, поражений и ничьих

## Как играть

1. Запустите игру: `python tic_tac_toe.py`
2. Выберите режим игры:
   - `1` для режима двух игроков
   - `2` для режима игрок против ИИ
3. Если играете против ИИ, выберите уровень сложности:
   - `1` для простого (ИИ делает случайные ходы)
   - `2` для сложного (ИИ использует алгоритм Минимакс)
4. Введите координаты для своего хода в формате "строка столбец" (оба значения от 1 до 3)
5. Игра продолжается до победы одного из игроков или заполнения поля (ничья)

## Правила игры

- Игроки поочередно ставят свой символ (X или O) на поле 3x3
- Первый игрок использует X и всегда ходит первым
- Игрок побеждает, если размещает три своих символа подряд по горизонтали, вертикали или диагонали
- Если все 9 клеток заполнены, а победитель не определен, игра заканчивается ничьей

## Технические детали

Реализация ИИ включает два подхода:

### Простой ИИ
- Делает случайные ходы по доступным клеткам
- Может быть обыгран с помощью базовой стратегии

### Сложный ИИ (алгоритм Минимакс)
- Реализует алгоритм Минимакс для оптимального принятия решений
- ИИ непобедим - он либо выигрывает, либо сводит игру к ничьей
- Оценивает все возможные будущие состояния игры для определения наилучшего хода
- Система подсчета очков:
  - +10 за победу ИИ
  - -10 за победу игрока
  - 0 за ничью

## Требования

- Python 3.x
- Только стандартная библиотека (без внешних зависимостей)
